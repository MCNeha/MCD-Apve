Media

The MEDIA module is responsible for all the media content that has been uploaded to the media endpoint and can be viewed on the template. There are four popular filters available at the top right of the screen:

  • LINECODE
  • LOGO
  • RESOURCE
  • CITY

A page navigation menu is also available at top of the screen, click any number to go to that specific page. Click the double backward arrow to go to the start page or the double forward arrow to go to the end page.

The MEDIA modules has the following user-defined filter options available to create:

Name
  1. Log in to the SITA AirportVision web portal.
  2. Click the MEDIA module in the left sidebar menu.
    • The Media screen displays.
  3. Click the icon next to Filters.
    • The Add new filter window displays.
  4. Click NAME.
    • The Name field displays.
  5. Click the Name field:
    1. Select a file name from the list that displays.
    2. Begin to type a file name and select the necessary one.
  6. Click CREATE.
    • Media results display.
    • Note: Click the icon next to the selected device to remove the filter.
 Extension
  1. Log in to the SITA AirportVision web portal.
  2. Click the MEDIA module in the left sidebar menu.
    • The Media screen displays.
  3. Click the icon next to Filters.
    • The Add new filter window displays.
  4. Click EXTENSION.
    • The Choose Extension field displays.
  5. Click the Choose Extension field:
    1. Select an extension type from the list that displays.
    2. Begin to type an extension type and select the necessary one.
  6. Click CREATE.
    • Media results display.
    • Note: Click the icon next to the selected device to remove the filter.
 Tag
  1. Log in to the SITA AirportVision web portal.
  2. Click the MEDIA module in the left sidebar menu.
    • The Media screen displays.
  3. Click the icon next to Filters.
    • The Add new filter window displays.
  4. Click TAG.
    • The Choose Tag field displays.
  5. Click the Choose Tag field:
    1. Select a tag from the list that displays.
    2. Begin to type a tag and select the necessary one.
  6. Click CREATE.
    • Media results display.
    • Note: Click the icon next to the selected device to remove the filter.
 Tag Value
  1. Log in to the SITA AirportVision web portal.
  2. Click the MEDIA module in the left sidebar menu.
    • The Media screen displays.
  3. Click the icon next to Filters.
    • The Add new filter window displays.
  4. Click TAG VALUE.
    • The Choose Tag and Value fields displays.
  5. Click the Choose Tag field:
    1. Select a tag from the list that displays.
    2. Begin to type a tag and select the necessary one.
  6. Click the Value field:
    1. Select a value from the list that displays.
    2. Begin to type a value and select the necessary one.
  7. Click CREATE.
    • Media results display.
    • Note: Click the icon next to the selected device to remove the filter.

Tags

Media content can include tags. Tag names and values can be added to each media type.

Add a tag or tag value to media
  1. Log in to the SITA AirportVision web portal.
  2. Click the MEDIA module in the left sidebar menu.
    • The Media screen displays.
  3. Click on a media card.
  4. Enter a Tag Name, if needed.
  5. Enter a Tag value, if needed.
  6. Click the icon to add them to the selected media.
