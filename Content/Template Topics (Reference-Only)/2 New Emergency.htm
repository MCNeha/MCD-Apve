Option 2 Emergency

Menu >SITA IDS Operations>EMERGENCY

The EMERGENCY module allows you to set an emergency message to broadcast to AirportVision-connected screens.

Emergency Options

Activate Emergency Broadcast Message

  1. Log in to the SITA AirportVision web portal.
  2. Click the EMERGENCY module in the left sidebar menu.
    • The Emergency Broadcast screen displays.
  3. Select one of the following emergency types:
    1. BOMB
    2. FIRE
    3. SHOOTER
    4. WEATHER
    5. TEST
    6. EMERGENCY
      • The default Broadcast Area and Message displays.
  4. Select a Broadcast Area.
  5. Optional: Edit the Message.
    • Note: The message must be 140 characters or less.
  6. Click ACTIVATE to activate the emergency message.
    • The Activate Emergency countdown displays for 3 seconds.
  7. Optional: Click CANCEL to cancel the activation.
    1. If CANCEL is clicked, the Emergency Broadcast screen displays.
    2. If CANCEL is not clicked, the Emergency Activated screen displays.

Edit an Activated Emergency Message

  1. Activate emergency broadcast message.
  2. Click EDIT EMERGENCY to edit the selected emergency or choose a different one.
  3. Click ACTIVATE to activate the edited emergency message.
    • The Activate Emergency countdown displays for 3 seconds.
  4. Optional: Click CANCEL to cancel the activation.
    1. If CANCEL is clicked, the Emergency Broadcast screen displays.
    2. If CANCEL is not clicked, the Emergency Activated screen displays.

Clear an Activated Emergency Message

  1. Activate emergency broadcast message.
  2. Click CLEAR EMERGENCY.
    • The Clear Emergency confirmation window displays.
  3. Click CLEAR EMERGENCY.
    • The Emergency Broadcast is cleared and all screens resume normal operations.
