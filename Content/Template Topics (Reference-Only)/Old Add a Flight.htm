Add a Flight

Menu >SITA IDS Operations>FLIGHTS

You can follow the below steps to add a new flight:

  1. Click on the Add Flight button. The Add Flight dialog box will display.

  1. Select Filter Type from the drop down .

    • Arrival

    • Departure

  2. Select Career Name from the drop down .

    Note: Fields marked with *required are mandatory fields.

  3. Enter yourFlight Number.

  4. Enter the Scheduled Time using the YYYY-MM-DD HH:MM format.

  5. Enter the Estimated Time using the YYYY-MM-DD HH:MM format.

  6. Enter the Actual Time using the using the YYYY-MM-DD HH:MM format.

    Note: If you click on the clock icon next to the scheduled/estimated/actual time, it will enter the current date and time.

  7. Select City from the drop down. The City Code will display in the text box below.

  8. Select CodeShares from the drop down menu.

  9. Enter your Flight number.

  10. Click on the + icon. The CodeShares and Flight number will display in the text box below.

  11. Select Status from the drop down menu.

  12. Select Gate from the drop down menu.

  13. Arrival Flight Only: Choose a baggage Claim number from the dropdown menu.

  14. Departure Flight Only: Choose a Check-in counter from the dropdown menu.

  15. Enter a Toast Message to display on the gate screen.

  16. Click Save.

    • Your new flight is saved.
