Introduction to APVE

What is APVe?

AirportVision Evolution (APVe) is an evolution of the existing APV system and is the next generation of SITA Visual Passenger Communication Systems.

It replaces dedicated display controllers with a built in OS/browser in the display screen.

APVe is part of the SITA Airport portfolio that allows airports to provide market leading FIDS to passengers.

APVe Template Options

APVe provides a Flight Information Display System (FIDS) with leading design templates. It currently supports the following templates:

  • Arrivals
  • Land-side Departures
  • Gate-side Departures
  • Check-in
  • Gates
  • Baggage Claim
  • Baggage Claim Summary
  • Static Media
  • Visual Messaging
