Log In Screen

Log in options to the SITA IDS Operations client module are dependent on your system configuration.

  1. You may have immediate access.

  2. You may have to access to the module using a valid user name and password.

    1. Enter a valid Username and Password.

    2. Click LOG IN.

    3. Choose a site from the Select a site dropdown.
      Note: The Select a site dropdown is only available when a user has access to more than one site.

The portal welcome screen displays after a successful log in.

Operations Client Dashboard

The following are the options available on the Operations Client dashboard

Search for a flight

  1. The portal welcome screen displays after a successful log in.
  2. Click the search field to the right of the magnifying glass icon.  

  Note:

  • Multiple criteria must be separated by a space.

  • The search field is not case-sensitive.

  • A wild card search can be added to any search criteria using an '*'. For example, *12 would be any field that contains 12.

  1. Enter any of the following search criteria: 
    1. Airline
    2. Flight number
    3. City
    4. Gate
    5. Claim
  2.  All flights matching the entered criteria displays.
    If no flights match the entered criteria, the message No results found, Try another search displays

Edit the displayed language or time zone

  1. The portal welcome screen displays after a successful log in.

  1. Click the arrow in the upper right corner next to your name.
    The Preferences window displays.

  1. Select the Time Zone:

  • Site

  • Local

  1. Select a Language:

  • EN — English

  • FR — French

  • ES — Spanish
    Note: English is selected by default.
