Introduction to APVE

What is APVe?

AirportVision Evolution (APVe) is the next generation of SITA Visual Passenger Communication System. APVe provides Flight Information Display System (FIDS) with leading design templates. APVe currently supports the following templates:

  • Arrivals
  • Land-side Departures
  • Gate-side Departures
  • Check-in
  • Gates
  • Baggage Claim
  • Baggage Claim Summary
  • Static Media
  • Visual Messaging

APVe is an evolution of the existing APV system. It focuses on replacing the dedicated display controllers with a built in OS/browser in the display screen.

The administration module's primary function is configuration in order to support the use of the system operationally.

New Key Features - Jamie

APVe is part of the Airport portfolio that allows airports to provide market leading FIDS to passengers. The new key features include:

  • Zone Management

  • New modern client technology

  • Improved Visual Messaging
