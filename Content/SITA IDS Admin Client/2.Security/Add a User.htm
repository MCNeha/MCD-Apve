Add a User

  1. Click on the Add User button.

    The Create New User window will display.

  1. Input your details in the Create New User window.

    Note: Fields marked with *required are mandatory fields.

  2. Select an Active or Inactive option

    Note: the Action option will allow users to log into and access the system.

  3. Click Save.

  4. Click Cancel, if you do not want to save the new User.
