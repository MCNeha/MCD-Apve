Filter Users

 

You can filter a User using all or any one of the following filter field name options.

Filter Field Name Filter Action

Name

Filter Users by Name
First Name

Filter Users by First Name
Last Name

Filter Users by Last Name
Email

Filter Users by Email
Role

Filter Users by Role
Note: Each filter field can be sorted alphabetically by clicking on the chevron icon next to the filter field name.
Note: The Actions column gives you three options .
You can edit a user, edit user API key or delete a user.
