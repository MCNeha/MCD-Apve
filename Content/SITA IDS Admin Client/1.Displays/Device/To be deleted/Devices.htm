Devices

The Devices area of the DEVICE module is divided into two sections:

  • Devices without a playlist are displayed in the top section
  • Devices with a playlist are displayed in the bottom section

The total number of devices currently available are displayed at the top of the screen. Filters are available to modify these displayed devices.

Device Filters

Three filter types are available to change the displayed devices.

Note: multiple filters may be applied at the same time.

The following device filter options are available:

Unique ID
  1. Log in to the SITA AirportVision web portal.
  2. Click the DEVICE module in the left sidebar menu.
    • The Device management screen displays.
  3. Click Devices.
    • All devices currently added display.
  4. Click the icon next to Filters.
    • The Add new filter window displays.
  5. Click UNIQUE ID.
    • The Unique ID field displays.
  6. Click Unique ID field:
    1. Select a unique ID from the list that displays.
    2. Begin to type a unique ID and select the necessary one.
  7. Click CREATE to add the filter.
    • Device results display. If there are none that match the selected filter, No Devices displays on the screen.
    • Note: Click the icon next to the selected device to remove the filter.
 Name
  1. Log in to the SITA AirportVision web portal.
  2. Click the DEVICE module in the left sidebar menu.
    • The Device management screen displays. 
  3. Click Devices.
    • All devices currently added display.
  4. Click the icon next to Filters.
    • The Add new filter window displays.
  5. Click NAME.
    • The Name field displays.
  6. Click the Name field:
    1. Select a name from the list that displays.
    2. Begin to type and select the necessary one.
  7. Click CREATE to add the filter.
    • Device results display. If there are none that match the selected filter, No Devices displays on the screen.
    • Note: Click the icon next to the selected device to remove the filter.
 Configuration
  1. Log in to the SITA AirportVision web portal.
  2. Click the DEVICE module in the left sidebar menu.
    • The Device management screen displays. 
  3. Click Devices.
    • All devices currently added display.
  4. Click the icon next to Filters.
    • The Add new filter window displays.
  5. Click CONFIGURATION.
    • The Configuration field displays.
  6. Click the Configuration field:
    1. Select a configuration from the list that displays.
    2. Begin to type a configuration and select the necessary one.
  7. Click CREATE to add the filter.
    • Device results display. If there are none that match the selected filter, No Devices displays on the screen.
    • Note: Click the icon next to the selected device to remove the filter.

Note: Add multiple filters by repeating above steps.

Device Playlists

In the Devices area of the DEVICE module, each device with or without a playlist displays in a card layout. For each device card:

  • The device name displays in bold blue text with the device unique ID listed underneath it.
  • A green ACTIVE button indicates the device status is powered on.
  • A black INACTIVE button indicates the device status is powered off.
  • A yellow NO ASSIGNMENT button indicates there is no playlist currently assigned to this device.
Assign a playlist to a device without a playlist
  1. Log in to the SITA AirportVision web portal.
  2. Click the DEVICE module in the left sidebar menu.
    • The Device management screen displays. 
  3. Click Devices.
    • All devices currently added display.
  4. Click the Assign a playlist field on a device without a playlist.
    1. Select a playlist from the list that displays.
    2. Begin to type a playlist and select the necessary one.
  5. Click the icon.
    • The selected playlist is assigned to the device and the device card moves to the Devices with a playlist section.
 Edit the assigned playlist of a device
  1. Log in to the SITA AirportVision web portal.
  2. Click the DEVICE module in the left sidebar menu.
    • The Device management screen displays. 
  3. Click Devices.
    • All devices currently added display.
  4. Hover your cursor over a device card to display the icon.
  5. Click the icon.
    • The Edit Device: [Device Name] window displays.
  6. In the Playlist Configuration section, click the Playlist field.
    1. Select a playlist from the list that displays.
    2. Begin to type a playlist and select the necessary one.
  7. Click SAVE.
    • The selected playlist is assigned to the device.

Device Details

Each device displayed in the Devices area of the DEVICE module include configurable Descriptors or details.

Edit device details
  1. Log in to the SITA AirportVision web portal.
  2. Click the DEVICE module in the left sidebar menu.
    • The Device management screen displays. 
  3. Click Devices.
    • All devices currently added display.
  4. Hover your cursor over a device card to display the icon.
  5. Click the icon.
    • The Edit Device: [Device Name] window displays.
  6. Edit or enter the following necessary fields in the Descriptors section:
    1. Model
    2. Manufacturer
    3. Operating System
    4. Zones
  7. Click SAVE.
    • The device details are saved.
  • Note: The Device name and unique ID are the identifiers for the selected device, these settings can not be changed.
