RegisterMe

Devices must be registered before they are available to edit and assign in the DEVICE module. Use the RegisterMe portal to register a new device.

Register a new display device
  1. Open an internet browser.
  2. Navigate to your Register Me portal:
    • Default site: < server URL >/apveclient/registerme
  3. Enter the Device Name.
  4. Click REGISTER button.
    • The message Device registered, waiting for configuration displays at the top of the screen in green background.
    • Note: This device displays as a new device card on the Devices screen of the SITA AirportVision web portal.
Remove a registration

Remove an existing registration from your local machine.

  1. Open dev tools in Chrome/Chromium with F12.
  2. Click on the application tab at the top.
    • Local Storage displays as a section on the left navigation panel.
  3. Delete the entry.
  4. Close the browser.
