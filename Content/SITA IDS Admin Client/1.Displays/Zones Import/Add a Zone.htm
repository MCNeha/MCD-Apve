Add a Zone/Device

  1. On the View Zones window, click the Add Zone button to select your required zone.

    • The Add Zone window will display.

      Tip: You have the choice to either add:
      - a Zone and it's associated Devices
      - add a Device to a Zone
      - add both a Zone and a Device at the same time.

  2. In the Zone Name text box input the name of your new Zone.

    Tip: You can also type in the Search field to find a specific Zone or Device to add.

  1. Enable the Protected Status button if required.

Tip: Enabling the Protected Status button prevents you from accidentally deleting the zone.

  1. Select a Child Zone: in the Edit Child Zones panel, click on a Child Zone to select it.

    • The selected text box will change to a purple color.

  2. To clear a Child Zone click on it again.

    Note: When you select a Child Zone, the devices attached to it will be added. If a selected Child Zone has a sub zone then it will be automatically selected too.

  3. Select a Device: in the Edit Child Devices panel, click on a Child Device to select it.

    • The selected text box will change to an orange color.

      Note: A device can be added to any Zone. When you select a Child Zone, devices attached to this zone will be automatically selected. If the Child Zone has a sub zone, then this sub zone and it's devices will be automatically selected.

  4. To clear a Child Device click on it again.

  1. Your Zone and Device selections will display in a list in the Selected Items display panel.

  2. Click Save.
