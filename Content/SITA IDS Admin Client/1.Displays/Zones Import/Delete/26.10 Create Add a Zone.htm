Add a Zone

  1. Click the Add Zone button to select your required zone or type in the Search text field to find your required Zone.

 

  1. Type in the name of your new Zone.

  2. Disable the Protected Status????? button the grey will display?

    Note: You can see the protected status button.
    Protected status button is to prevent users from an extra step to prevent accidental deletion.

  3. In the Edit Child Zones panel, click on a Child Zone to select it. The text box will display in purple.

    Note: When you select a child zone, the devices attached to it will be added. If a selected Child Zone has a sub zone then it will be automatically selected.
