Introduction to Zones

TheVISUAL MESSAGING module allows you to create textual messages to be played on a specific group of devices, known as a Zone.


A zone is a 'display group'. A message can be sent to every display device in that zone.

Created zones can include other zones or devices. Created zones need to be editable and removed from within the Admin Client. Zone features include the options to prevent access, create, edit or delete a zone.
