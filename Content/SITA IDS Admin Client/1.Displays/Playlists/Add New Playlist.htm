Add a Playlist

  1. Click on the Add New Playlistbutton.

    The Add New Playlist window will display.

  2. Select a template from the drop-down list to add a new template.

  3. Note: the Playlist has a Template Interval of 20 seconds. This interval determines how often the templates rotate between each other.

  4. Click on the + icon to add a new playlist as per the template selected.

  5. Click Save.

  6. Click Cancel, if you do not want to save the playlist.
