Filter Playlist

You can filter a playlist using either all or any one of the following filter text field options:

  1. Enter the Playlist Name to filter as per playlist name.
  2. Click Content drop-down menu to select the content.
  3. Click Template Type drop-down menu to select the Template type.
  4. Filtered screen will display.
  5. You can remove the filters applied by clicking on the x icon in the field.
