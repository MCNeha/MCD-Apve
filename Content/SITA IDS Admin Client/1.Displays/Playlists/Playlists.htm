Playlist Overview

 

ThePlaylistssetting screen allows you to manage device behaviors with configurable playlists.

The Playlists area of the Device module of the admin client allows you to manage settings and create new playlists.

You can:

  • Add New Playlist

  • Delete Playlist

  • Edit Playlist

  • Filter Playlist
