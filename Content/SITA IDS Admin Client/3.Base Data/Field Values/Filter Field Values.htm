Filter Code

You can filter the code field value, to filter the field click on the icon. The following filter box is displayed.

  1. Select the condition from the drop-down list. The following options are available for the condition:

  • Equals

  • Not Equals

  • Contains

  • Starts with

  • Ends with

  • Does not contain

  • Null

  • Is not Null
