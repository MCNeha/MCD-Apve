Edit a Translation

  1. Click on the icon against the translation that you want to edit.
    Edit Translation window will display.

  1. You can not edit the greyed out fields.

  2. Edit the fields you want to change.

  3. Click Save.

  4. The following message is displayed on the screen.

  5. Click Cancel, if you cancel the edit action.

Note: Grayed out fields such as Application, Area and Key are not configurable.
