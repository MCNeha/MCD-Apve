Consistent Template Features

The templates have consistent features included that are designed to aid in readability and usability.

Flight Status

When status appear on a template, they appear with the following attributes:

  • Status updates — a blinking animation displays to alert passengers of updates. Animations can be turned on or off.
  • Status outside of range — On Time theme displays.
  • Status attributes

 Paging and Bank Display

Templates can display on single screens, banks, grids, or any combination and can be set in the configuration settings.

Any single page template can iterate through multiple pages of flights. This is useful in single screen deployments or when the number of flights exceeds a number of screens in bank. Paging indicators display at the bottom of the template to indicate to a user that there are multiple pages.

No Connection Displays

When the client/server connection is broken and the No Connection Timeout threshold is met, an error message displays stating that flight information is delayed.
