Why provide administrative client rewrite?

The Administrative Client Rewrite is a new modern client written using modern web technologies, and designed by in-house SITA UI Experts. It runs on almost any modern device from mobiles, tablets and desktops. It provides more accessibility to busy airport operations staff allowing them to deliver the most up to date flight information to passengers.

This online help outlines the following core features of the new administrative client:

  • Departure Gate administrative module
  • Flight tracking module
  • Media module to maintain images displayed on templates for airlines, destinations and marketing
  • Device and configuration module to manage devices and display configurations
  • Check-In module for baggage check-in counter display management.
