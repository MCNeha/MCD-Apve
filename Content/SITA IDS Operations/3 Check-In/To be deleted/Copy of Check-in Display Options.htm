Display Options for Check-in Counter

A Check in counter screen displays media and announcements regarding a flight. You can configure the display of media and announcements using a number of design options.

These Check In design options allows you to customize the design and display of media and announcements of your check-in counter.

The range of items to display on a check in counter include an Airline Logo,a Service Type, an Announcement, display Eligible Flights,

  1. Select a Check-in counter location

  1. Click the Check-in location toggle button to open or close the check-in counter.
    The check-in counter will display OPEN or CLOSED depending on your choice.

Update the Check-in Counter Style

  1. Click Style from the Check-in design section.
  2. Select the Colors for display:
    1. Select the Background color.
    2. Select the Text color.
  3. Select the Background image:
    1. Click the Change link to select the Background image.
      The Select content window displays.
    2. Select tag name.(This is an Optional field)
    3. Click Add Filter to apply the filter.
    4. Select a check-in logo.
    5. Click SELECT.
  4. Click Publish.
    The updated check-in counter style is saved and published.
  5. Click Revert to reset back all check-in design settings to their last publication.

Frequently Asked Questions

Q 1 - How can I select/assign multiple check-in counters for the same flight while adding new flight?

This feature is not currently available in the client, but will be available in the future.

 Q 2 - How can I select a current flight for a check-in counter?

Check-in counters will only display flights that are assigned to that check-in.

 Q 3 - How can I verify a check-in counter design?

Check-in templates can be verified in a web browser.

  1. Click the Check-in name.
    • The Select Check-in window displays.
  2. Click the necessary check-in location name.
  3. Click the Style menu in the Check-in design section.
