Design a Style

Customize the look and appearance of the display of the check-in counter:

  1. Click Style from the Check-in design section.
  2. Select Colors for display:
    1. Select the Background Color.
    2. Select the Text Color.
    Tip: Click the dropdown menu to select your new Background Colorand the Text Color.
    You can clear the Background Colorand Text Color by using the Unset option.
  3. Select the Background image:
    1. Click the Change link to select a Background Image.
      The Select content window will display.
    2. Select tag name.(This is an Optional field)
    3. Click Add Filter to apply the filter.
    4. Selectthe Background Image
    5. Click Select
  4. Click Publish.
    The updated check-in counter style is saved and published.
  5. Click Revert to reset back all check-in design settings to their last publication.
