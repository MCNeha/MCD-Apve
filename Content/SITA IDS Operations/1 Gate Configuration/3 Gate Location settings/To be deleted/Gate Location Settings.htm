Gate Location Settings

Neha Is this an Ops function isnt it? What action needs to be taken and where do you get these values?

 

The Gate Location Settings screen allows gate agents to update the gate location settings for gates.

View the Gate Location Settings

  1. Click the Gate Location Settings icon.
    The Gate Location Settings screen displays with all existing gate location settings listed.

Edit a Gate Location Settings Value

  1. Click the Gate Location Settings icon.
    The Gate Location Settings screen displays with all existing gate location settings listed.
  2. Click the icon next to an existing gate location setting.
  3. Edit the Value, if needed.
  4. Click Save.
    The updated gate location setting value is saved.
