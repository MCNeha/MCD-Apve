Update Gate Status

  1. Click the GATE module in the left sidebar menu.
    • The Gate Operations screen displays.
  2. Click the gate number or the icon next to it.
  3. Choose a departure gate number from the list.
    • The selected gate displays in the field with local current time and date.
  4. Click the current status or the downward facing arrow beside it.
  5. Select one of the following options:
    1. CANCELED
    2. ON TIME
    3. DELAYED
    4. SEE AGENT AS SOON AS POSSIBLE
    5. GATE OPEN
    6. GO TO GATE
    7. NOW BOARDING
    8. LAST CALL
    9. FLIGHT CLOSED
    10. DEPARTED
    11. GATE CLOSED
    12. OT
    13. AWAIT GATE
    14. BUS TRANSFER
Note: When a boarding sequence is assigned to a gate, the gate status option is disabled.
Note: If there is an operational flight available, this option is displayed on the bottom of the boarding sequence screen. The DISPLAY NEXT FLIGHT button is also enabled. If there is no operational flight available, theNo flight displaying message is displayed on the boarding sequence screen.
