Gate Operations

The Gate Operations window allows you to start or continue an operation at the gate.This Gate operations configuration is intended to edit, add or update a sequence.

You can view upcoming flights at gates and assign and clear flights from a current gate. You can also change the current gate remark.

In your Gate Operations window you can also assign a boarding sequence to a flight.

This action requires four steps:

  1. Select a gate location.

  2. View the flights assigned to your selected gate location.

  3. Assign a flight to this gate location.

  4. Assign a boarding sequence to the flight.

Gate Operations Panels Action

1 Departure Gate Panel

  1. To select a departure gate location, click on the departure gate number or on the chevron icon.

    A Select Gate list and a Search option will display.

  1. Select a departure gate number from the drop down list.
    OR
  2. Search for a departure gate number by inputting the required gate number.
Note: The selected gate displays with the local current time and date.

2 Flight Panel
  1. If no flights are associated with your selected gate, the text No flights will display.
  2. If Departure flights are assigned to your selected gate they will be displayed.
  3. Double click your departure flight to open the flight details
  4. A Confirm display dialog box displays.

  5. Click Display to display the flight details and assign the flight to the selected gate.

    The flight details will display.

  6. Click Cancel, flight details will not be displayed.

  7. Press the Clear Flight button to clear the flight details, remove the flight from this gate and clear all related boarding sequence, gate status, messages data.

    Note: If the flight time has changed, the new time will display in red below the scheduled flight time.

3 Boarding Sequence Panel

  1. Click Select Sequence to assign one or more sequences to the flight. A pop up model will display with multiple available sequences if applicable. See more information on boarding sequences.

  1. Select a named sequence to view all available sequence steps assigned to it.
  2. Click Assign to assign the sequence to the gate/flight.
  3. Click Start to start the sequence.
  4. Click Next to go to next step.
  5. Click Replay to restart a boarding sequence step
  6. Click Complete to complete the entire boarding sequence process for the gate.
See more information on the Boarding Complete process

(After clicking the COMPLETE button, the text Boarding Complete displays in green with a checked circle on the boarding sequence screen. The READY FOR DEPARTURE message displays at the top of the boarding sequence section with a green background to indicate the operational flight is ready for departure. This is the Drop-down text)
  1. Click Continue to refresh the current gate
  2. Cycle through or replay a boarding sequence
  3. Click here for information to select a boarding sequence

4 Flight Status Announcement Panel
  1. If a Boarding Sequence has not been assigned to a flight, you can chose a Flight status from the drop down Announcement menu.
  2.  If a Boarding Sequence has been assigned to a flight the announcement remark panel automatically updates in synchronicity with your Boarding Sequence.
  3.  Press Start to start the Boarding Sequence.

  4. Click Next to cycle through the Boarding Sequences until the sequence is complete. As you complete the steps the text will change from Blue to Green.

    Note:
    Remark options include: Cancelled, Delayed, Departed, Flight closed, Now Boarding, On time, See Agent as soon as possible, Off Time.

  5. Press Complete.The Boarding Sequence is now complete and the message panel will display Off Time.

  6. Press Continue and the flight information will disappear.

  7. Optional Step: Select an announcement.
