Boarding Sequences Options

The Boarding Sequences screen allows you to perform the following actions:

  • View all sequences

  • Add a new sequence

  • Edit a sequence

  • Delete a sequence

View all sequences

  1. Click the Boarding Sequences icon.
    • The Boarding Sequences screen displays with all existing boarding sequences listed.
  2. Click the icon to view sequences on next page.
  3. Click the icon to view sequences on previous page.  
  1.  
    • Note: A list of 15 boarding sequences are displayed on the screen by default.

Add a new sequence
  1. Click the GATE module in the left sidebar menu.
  2. Click the Boarding Sequences icon.
    • The Boarding Sequences screen displays.
  3. Click Add New.
    • The New boarding sequence window displays.
  4. Enter a descriptive Name for the sequence.
  5. Select an Airline or type an airline name directly into the Select aircraft field. Leave blank for all airlines.
  6. Click the icon to add the selected airline.
  7. Select an Aircraft or type an aircraft type directly into the Select aircraft field. Leave blank for all aircrafts.
  8. Click the icon to add the selected aircraft.
  9. Select a gate or type a gate number directly into the Select gate field. Leave blank for all gates.
  10. Click the icon to add the selected gate.
  11. Repeat steps 6 - 11 until the necessary Scope details are selected.
  12. Select a sequence step or type the step directly into the Add step field.
  13. Click the icon to add the step.
  14. Repeat steps 13 - 14 until all necessary steps are added.
  15. Click and move steps up or down to reorder them.
  16. Click SAVE.
    • The new boarding sequence is saved.
Edit a sequence
  1. Click the GATE module in the left sidebar menu.
  2. Click the Boarding Sequences icon.
    • The Boarding Sequences screen displays.
  3. Click the icon next to an existing boarding sequence.
    • The Update boarding sequence window displays.
  4. Edit the Name of the boarding sequence, if needed.
  5. Add or remove any of the following Scope details:
    • Airline
    • Aircraft
    • Gates
  6. Add or remove any Sequence steps.
  7. Click and move steps up or down to reorder them.
  8. Click Save.
    • The updated boarding sequence is saved.
 Delete a sequence
  1. Click the GATE module in the left sidebar menu.
  2. Click the Boarding Sequences icon.
    • The Boarding Sequences screen displays.
  3. Click the icon next to an existing boarding sequence.
    • The Confirmation window displays.
  4. Click DELETE.
    • The selected boarding sequence is deleted.
