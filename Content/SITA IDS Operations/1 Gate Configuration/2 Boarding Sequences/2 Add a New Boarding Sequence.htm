Add a Sequence

  1. Click Add New.
    The New boarding sequence window displays.
  2. Enter a Name that briefly describes the new boarding sequence.
  3. Select/add an Airlinethat may use this boarding sequence or type an airline name directly into the Airline field. Leave blank for all airlines.
  4. Select an Aircraft that may use this boarding sequence or type an aircraft type directly into the Select aircraft field. Leave blank for all aircrafts.
  5. Select a Gate that may use this boarding sequence or type a gate number directly into the Select gate field. Leave blank for all gates.
  6. Select a sequence step or type the step directly into the Add step field.
  7. Repeat step 6 until all necessary sequence steps are added.
  8. Add, change, move and delete steps to refine the boarding sequence.

Tip: To reorder a sequence, hover your cursor over a sequence step and move it up or down.

  1. Click SAVE.
    The new boarding sequence is saved.
