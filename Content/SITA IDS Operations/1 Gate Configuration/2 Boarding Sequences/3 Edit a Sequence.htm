Edit Sequence

  1. Click the icon next to an existing boarding sequence.
    The Update boarding sequence window displays.
  2. Edit the Name of the boarding sequence, if needed.
  3. Add or remove any of the following Scope details:
    • Airline
    • Aircraft
    • Gates
  4. Add or remove any Sequence steps.

  5. Click and move the sequence steps up or down to reorder them using the arrow keys.

  6. Click Save.
    The updated boarding sequence is saved.
