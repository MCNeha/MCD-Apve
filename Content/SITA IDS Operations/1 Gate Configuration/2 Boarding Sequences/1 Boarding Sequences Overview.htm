Boarding Sequences

The Boarding Sequences screen displays all existing boarding sequences in association with an airline, an aircraft and a gate. It allows you to edit or create a boarding sequence for use at a gate.

Note: To begin and complete a boarding sequence, the system must be enabled to send updates to the Gate template displayed to passengers on FIDs screen.
