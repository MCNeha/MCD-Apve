Filter a Flight

You can filter a flight using either all or any one of the following filter text field options:

  1. In the FROM text box, click on the Calender icon to select a date and a time.
  2. In the TO text box, click on the Calender icon to select a date and a time.
  3. Click Airline drop-down menu to select the airline.
  4. Click Cities drop-down menu to select the city.
  5. Click Claim drop-down menu to select the baggage claim.
  6. Click Check-in drop-down menu to select the check-in. 
  7. Click on Apply to apply the filters selected.
    The filtered data will display.
  8. When the filters are applied to the flight, the remove filter icon will appear next to the Apply button.
  9. Click on the icon to remove the applied filters.
