Emergency

The EMERGENCY broadcast module allows you to activate an emergency message to be broadcast to SITA Information Display System connected screens.

Emergency Message Display Options

The following message options are available.

Activate Emergency Message

  1. Click the EMERGENCY module in the left sidebar menu.
    • The Emergency Broadcast screen displays.
  2. Select one of the following emergency types:
    1. BOMB
    2. FIRE
    3. SHOOTER
    4. WEATHER
    5. TEST
    6. EMERGENCY
      • The default Broadcast Area and Message window displays.
  3. Select a Broadcast Area
    The default is set to ALL DEVICES

  4. Optional: Input a new message or you can edit the current Message.
    • Note: The message must be 140 characters or less.
  5. Click Activate to activate the emergency message in the selected Broadcast Area.
    • The Activate Emergency countdown displays for 3 seconds. It also provides a summary of the emergency message and where it was displayed.
  6. Optional: Click Cancel to cancel the activation.
    1. If CANCEL is clicked, the Emergency Broadcast screen displays.
    2. If CANCEL is not clicked, the Emergency Activated screen displays.
Note: An error message will display if the message has not been displayed.
