Filter Messages

You can filter Visual Messages by following the steps below:

  1. In the FROM text box, click on the calendar icon to select the message start date and time.

    Click Ok.

    • The start date and start time will display in the FROM field.

  2. In the TO text box, click on the calendar icon to select the end date and time of the message.

    Click Ok.

    • The end date and endtime will display in the TOfield..

  3. Select Active or Inactive from the drop-down list. 

    Note: If you select Active this selection will display messages that are active in the system during the time displayed and are colored green. An Inactive selection will display inactive messages (no longer active) during the time displayed, these messages are colored Grey.

  1. Click on Zones to select the required zone(s) where the message will be played.

  1. Click on UPDATE.

    The filtered messages will display.

Tip: You can temporarily deactivate a message. It will remain within the time range you set originally.
