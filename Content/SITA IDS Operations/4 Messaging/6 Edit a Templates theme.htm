Edit a Theme Template

The following Templates theme attributes can be adjusted:

  • Title or subtitle

  • Clock (can be turned off /on)

  • End time (can be sorted)

  • Pagination (can set to one page)

    Note: Pagination - this will select depending the amount of text being shown.

     

  • Font size

  • Individual colours

  • Radius (can be turned it on/off)

     

  • Background image of the header

  • Logo Type
