Edit Message

You can edit a Visual Message by following these steps:

  1. Double click on the message you want to edit.

  2. The message is displayed in the edit message window.

  3. Edit your message as required.

  4. You can also edit the zones of the message.

  5. You can either select more zones or de-select the selected zones of the message.

  6. Click on Publish.

  7. The Success toast message displays on the screen.

  8. The edited message can be seen on the Message Dashboard.
