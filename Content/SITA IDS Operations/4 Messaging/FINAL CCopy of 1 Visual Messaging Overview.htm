Visual Messaging

Visual Messaging allows you to display specific messages in specific Zones.

Visual Message Options

You can create, edit, filter and sort current (active) messages and past (inactive) messages.

You can also select what specific Zone(s) to display these messages in.

Edit Visual Message

You can edit a Visual Message by following these steps:

Double click on the message you want to edit.

The message is displayed in the edit message window.

Edit your message as required.

You can also edit the zones of the message.

You can either select more zones or de-select the selected zones of the message.

Click on Publish.

The Success toast message displays on the screen.

The edited message can be seen on the Message Dashboard.
