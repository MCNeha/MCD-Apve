Create Message

You can create a Visual Message by following these steps:

  1. Click on the +New Message button on the topmost right corner to create a new message.

    • The following screen will display.

  1. Type your message into the Message panel.

    Note: Your message character limit is 500.

  1. In the Zones panel, select your required Zone(s) to display a visual message

    Note: the scheduling default is set to Immediate . The schedule starts immediately and is active for an hour.

 

  1. Schedule your message display time by clicking on Scheduled. You can set the FROM: (begin) and TO: (end) time period or set Never End for the time period for your messages to be displayed.

    Note: For your message to run continuously check the Never End checkbox.

  1. Click on PUBLISH to publish the message.

  2. The SUCCESS message displays on the screen.

  1. The published message can be seen on the Message Dashboard.
