    Update Gate Status

    Query to the PO

    Can we Delete all this? -it's about a Flight status- it's not a Gate status

     

    Is it a flight status on the gate

    1. Click the gate number or the icon next to it.
    2. Choose a departure gate number from the list.
      • The selected gate displays in the field with local current time and date.
    3. Click the current status or the downward facing arrow beside it.
    4. Select one of the following options:
      1. CANCELED
      2. ON TIME
      3. DELAYED
      4. SEE AGENT AS SOON AS POSSIBLE
      5. GATE OPEN
      6. GO TO GATE
      7. NOW BOARDING
      8. LAST CALL
      9. FLIGHT CLOSED
      10. DEPARTED
      11. GATE CLOSED
      12. OT
      13. AWAIT GATE
      14. BUS TRANSFER

    Note: When a boarding sequence is assigned to a gate, the gate status option is disabled.

    Note: If there is next operational flight available, the suggestion is then displayed on the bottom of the boarding sequence screen with DISPLAY NEXT FLIGHT button enabled. Otherwise No flight displaying message is displayed on the screen.

    Cycle through or replay a boarding sequence

    1. (Click the gate number or the icon next to it.

    1. Choose a departure gate number from the list.
      • All departure flights assigned to the selected gate display under the selected departure gate number.
    2. Select a flight.
      • The boarding sequence displays to the right.
    3. Click Next on a boarding sequence step to cycle through the step.
    4. Click Replay to restart a boarding sequence step.
